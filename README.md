# Expert Finance

## Loyiha haqida

Expert Finance — bu 2 nafar sherik tomonidan birgalikda ishlatiladigan Android ilova.

Ilovaning asosiy vazifasi:

- Daromadlarni boshqarish
- Xarajatlarni boshqarish
- Hisoblarni yuritish
- Olinadigan pullarni nazorat qilish
- Beriladigan pullarni nazorat qilish
- Hisobotlarni shakllantirish

---

## Foydalanuvchilar

Tizimda faqat 2 ta foydalanuvchi ishlaydi.
